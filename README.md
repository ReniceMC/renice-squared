<p><strong><font size="6">Renice²</font></strong></p>

A lightweight edit of Renice Cubed that removes advantage-giving features while keeping performance improvements, QoL, and customization built around the way you play. It keeps the game clear, smooth and comfortable, without conflicting with public server rules against unfair client-side advantages.
